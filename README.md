# community-blood-donor-finder
A modern React application built with Vite and Tailwind CSS that enables users to filter and request blood donors from an API dataset. Demonstrates derived state management, conditional rendering, and dynamic UI updates.

# Project Overview


This project provides a simple interface to:

View donor information

Filter donors by blood group

Search donors by city

Sort donors by availability

Send and withdraw requests

The donor data is fetched from a public mock API.


# Tech Stack

React (Vite)

JavaScript

Tailwind CSS

Public API: https://jsonplaceholder.typicode.com/users
